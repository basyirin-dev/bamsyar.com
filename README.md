# bamsyar.com

Personal research site and blog. Built with Astro 5, Tailwind CSS v4, deployed on Cloudflare Pages.

Live site: https://bamsyar.pages.dev

## Stack
- Framework: Astro 5
- Styling: Tailwind CSS v4 + @tailwindcss/typography
- Math: remark-math + rehype-katex (build-time)
- Deployment: Cloudflare Pages (auto-deploy from main)

## Development
npm install
npm run dev

## Build
npm run build
